게임 내부의 모든 오브젝트 클래스는 CGameObj 클래스를 상속받는다.

멤버 변수
-모든 객체에서 공통적으로 포함하는 특성인 좌표와 RECT 영역을 ObjectInfo 구조체로 받아온다.


멤버 함수
-Initialize() : 순수 가상 함수이며 이 클래스들을 상속 받은 모든 클래스들은 각자의 초기화 함수를 구현해야 한다.

-Update() : 순수 가상 함수이며 이 클래스들을 상속 받은 모든 클래스들은 각자의 갱신 함수를 구현해야 한다.

-Render() : 순수 가상 함수이며 이 클래스들을 상속 받은 모든 클래스들은 각자의 렌더 함수를 구현해야 한다.

-Release() :	순수 가상 함수이며 이 클래스들을 상속 받은 모든 클래스들은 각자의 해제 함수를 구현해야 한다.
				이 함수는 소멸자에서 단 한번만 호출된다.	

-UpdateRect() : 객체의 RECT를 객체의 너비와 높이에 걸맞게 조절한다.
